# AGENTS.md

This is the **standards** repository for the Kronos/Equisoft organization. It contains the canonical
Architecturally Significant Requirements (ASRs), shared tooling configurations, and coding standards
used across all repositories.

---

## Organization-Wide Standards

You **must** read and follow the organization-wide standards defined in
[`docs/instructions/asr.instructions.md`](docs/instructions/asr.instructions.md).

These instructions cover languages & stack, project setup, repository & versioning, code quality & CI,
dependency management, and feature flags. They apply to **all** repositories in the organization.

### Downstream Usage

Other repositories in the organization should reference these standards in their own `AGENTS.md` by
including a URL to the raw file hosted in this repository:

```markdown
You **must** read and follow the organization-wide standards defined at:
https://raw.githubusercontent.com/kronostechnologies/standards/master/docs/instructions/asr.instructions.md
```
